Teaching workflow

Useful before, during, and after class.

Set a common reference

Confirm the learner’s Sa and use the same swara vocabulary on both sides of a remote or in-person lesson.

Demonstrate steadiness

Show how a held note moves around its centre without reducing musical learning to a score.

Shape a home drill

Ask learners to repeat a target note or phrase using today’s live feedback and session summary.

Review together

Use the session summary or a recording shared by the learner as a starting point for feedback.
