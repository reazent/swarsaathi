## Full description

SwarSaathi is an Indian music learning app. SwarPractice helps you set your Sa, sing, and instantly see the swara, saptak, pitch, and tuning feedback.

Features in this release:

- Real-time swara and saptak detection
- Timed drills and free practice mode
- Offline tanpura for available pitches and tunings
- Continuous reference-note playback while you sing
- Optional metronome for rhythm support
- Optional local session recording and sharing
- On-device microphone processing

Tabla accompaniment is not included in this release.
