# v1.1 iTablaPro benchmark checklist

Use this checklist to compare SwarPractice v1.1 against iTablaPro-style practice expectations using the same pitch, room, phone position, and sustained-note durations.

## Setup

- Device: iPhone used for SwarPractice testing.
- Reference app: iTablaPro.
- Pitch: test at C3 first, then one lower and one higher pitch if time allows.
- Distance: phone 20-30 cm from mouth for singing tests.
- Environment: quiet room, no fan/AC near mic.
- Accompaniment: test without accompaniment first, then tanpura, then metronome.

## Sustained-note tests

Run each note for 5s, 10s, and 20s:

- Soft voice Sa.
- Medium voice Sa.
- Soft voice Pa.
- Medium voice Pa.
- Soft voice Ma.
- Slightly sharp Sa.
- Slightly flat Sa.

For each run, record:

- Does SwarPractice detect the note at low volume?
- Does the visual wave stay calm enough to read?
- Does the steady band make steadiness obvious?
- Does the score feel fair for the actual singing?
- Does Expert remain challenging without punishing natural vibrato/wobble too much?
- Is response latency acceptable compared with the reference app?

## Accompaniment checks

- Tanpura starts only after user gesture/audio permission.
- Tanpura follows the shared Your Pitch selector.
- Sa-Pa and Sa-ma modes load the expected recordings.
- Metronome starts/stops cleanly.
- BPM changes update the next beat interval.
- Cycle accents are audible for Teentaal, Keharwa, Dadra, Ektaal, and Rupak.
- Tabla control remains disabled until real loops are uploaded.

## Acceptance notes

SwarPractice does not need to clone iTablaPro. It should feel credible for daily learner practice: soft singing is detected, the visual guide is readable, scoring is encouraging but honest, and accompaniment does not destabilize pitch detection.
